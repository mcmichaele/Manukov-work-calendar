# Manukov Work Calendar

Auto-updating work schedule calendar for Manukov.

## 📅 Subscribe to the Calendar

Paste this URL into any calendar app:

```
https://raw.githubusercontent.com/mcmichaele/Manukov-work-calendar/main/manukov.ics
```

### Apple Calendar
Settings → Accounts → Add Account → Other → Add Subscribed Calendar → paste URL

### Google Calendar
Other Calendars → From URL → paste URL

---

## 📸 Updating the Calendar (For Admins)

1. Take a screenshot of the new monthly schedule
2. Upload it to the `schedules/` folder in this repo
3. That's it — the calendar updates automatically within ~1 minute

The automation reads the schedule image using AI, finds all Manukov shifts, and appends them to the calendar file.
